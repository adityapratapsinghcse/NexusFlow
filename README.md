# NexusFlow

### Smart Events. Seamless Execution.

NexusFlow is an AI-powered event management platform designed for colleges and organizations to efficiently manage events, registrations, venue allocation, attendance, notifications, and administrative workflows.

---

# 🚀 Features

## 👨‍💼 Admin Features

* AI-based venue allocation
* Event approval & publishing
* Attendance tracking
* Ban management system
* Broadcast notifications
* Audit logs
* Event cancellation management
* Venue reassignment
* Student management dashboard
* Feedback analytics

---

## 🧑‍🏫 Organizer Features

* Create and manage events
* Track registrations
* Publish passes
* View attendee details
* Reassign venues
* Manage schedules

---

## 🎓 Student Features

* Register for events
* View event status
* Receive notifications
* Access digital passes
* Submit event feedback

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/nexusflow.git
cd nexusflow
```

---

## 2. Create Virtual Environment

```bash
python -m venv venv
```

---

## 3. Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5. Run Database Migrations

```bash
python manage.py migrate
```

---

## 6. Start Development Server

```bash
python manage.py runserver
```
